# Web Development Workspace

This workspace is configured for the **Web Development team**.

## How to use this folder

1. **Review Stack:** Read `CLAUDE.md` for Next.js 14 App Router and TypeScript standards.
2. **Provide Input:** Paste a mockup image (Command+V) or write a natural language description (e.g., `sample_feature_request.txt`).
3. **Trigger Workflow:** Use the defined commands to generate full-stack components.

**Example Prompts:**
- `/build-component [paste image]`
- `/edge-cases "user dashboard profile settings"`
- `Update the state management logic per sample_feature_request.txt`

# Quarterly Sales Report Generator

Welcome to the Finance & HR workflow. This directory helps you generate plain-text or CSV style quarterly reports from raw data.

## Usage Instructions

1. **Review Data:** Ensure `sample_sales.csv` contains the correct quarter's transactions.
2. **Open Antigravity/Claude Code:** Inside this folder.
3. **Run the Prompt:** Type the following exact prompt:
   > "Plain-text workflow: 'Query sales data, produce quarterly Excel report' using MCP-style simulation. Base it on `sample_sales.csv`."
4. **Auto-accept:** Remember to use `Shift+Tab` to auto-accept safe generation tasks.
5. **Output:** Claude will generate a structured summary or simulate building the Excel document structure for you.

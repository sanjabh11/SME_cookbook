# Legal & Compliance Workspace

This workspace is configured for the **Legal and Compliance team**.

## How to use this folder

1. **Review Rules:** Read `CLAUDE.md` to understand the available workflows (accessibility audits, compliance trackers, phone-tree routing).
2. **Provide Input:** Place any document, URL, or code snippet you want audited in this folder (e.g., `sample_accessibility_snippet.html`).
3. **Ask Claude:** Use plain English to trigger a workflow.

**Example Prompts:**
- `Run an accessibility audit on sample_accessibility_snippet.html`
- `/compliance-tracker`
- `/legal-routing`

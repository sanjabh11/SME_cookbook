# Prototyping from Image Workspace

This is a cross-team workflow accessible to **all staff**.

## How to use this folder

1. **Review Rules:** Read `CLAUDE.md` to see the supported workflows (UI HTML, ROS2 Node, Product Spec, Quick Tools).
2. **Provide Image:** Either drag-and-drop an image into Claude Code or use `Command+V` to paste a screenshot.
3. **Trigger Workflow:** Tell Claude exactly what to do using a slash command or plain English.

**Example Prompts:**
- `Paste a Figma screenshot + /prototype-ui`
- `Take a photo of a whiteboard + /prototype-ros`
- `Take a sketch on a napkin + /prototype-spec`
